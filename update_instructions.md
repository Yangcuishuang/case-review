# 更新网页数据指南

## 快速更新流程

### 步骤 1：准备新数据
将新的案例数据准备好（JSON 格式或 Excel 文件）

### 步骤 2：联系 AI 助手
把新数据发给 AI 助手，说明：
- 需要更新哪些案例
- 新增还是修改
- 是否需要调整其他内容

### 步骤 3：获取更新文件
AI 助手会生成新的 `index.html` 文件

### 步骤 4：上传到 GitHub
1. 访问：https://github.com/Yangcuishuang/case-review
2. 点击 `index.html` 文件
3. 点击右上角 **编辑**（铅笔图标）
4. 全选并删除旧内容（Ctrl+A / Cmd+A）
5. 粘贴新内容
6. 点击 **提交更改**

### 步骤 5：等待生效
等待 1-2 分钟，访问 https://yangcuishuang.github.io/case-review/ 查看更新

---

## 注意事项

- 每次更新会覆盖整个文件
- 建议更新前备份旧数据
- GitHub Pages 有缓存，可能需要等待几分钟
